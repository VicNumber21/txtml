Introduction

The project was inspired by js sequence diagrams project. The main goal of the project is to extend this approach to another UML diagrams. More details can be found in README file in the project repository.

This is Proof-of-Concept version. It can't draw a real UML for the moment. It is a goal for the next milestone.

Live demo

Below you can find live demo of TxtML. By default it demonstrates TxtML representation of Decorator design pattern. Feel free to modify the representation to notice changes on the diagram.

Syntax of TxtML can be found in README file. Note that that a red color of text in the text area means that there is a syntax mistake in diagram description.
